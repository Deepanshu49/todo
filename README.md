# Simple To‑Do App

A tiny local-only to-do app (no server required). Files are in the `todo/` folder.

How to run

- Open `todo/index.html` in your browser (double-click or "Open File")
- Or serve the folder with a simple static server (optional)

Features

- Add tasks (press Enter or click Add)
- Mark tasks complete with checkboxes
- Delete tasks
- Clear completed or clear all
- Tasks are saved in `localStorage` (persist between reloads)
